# Hero Video

Drop a file here named `hero-ocean.mp4` and it will become the homepage background video.

## Recommended specs
- Format: MP4 (H.264)
- Resolution: 1920x1080 (1080p)
- Duration: 10–25 seconds, looping friendly
- File size: 2–5 MB (compress if larger)
- Content: slow, cinematic — drone shots, yacht cutting water, sunset ocean

## Free sources (no attribution required)
- https://www.pexels.com/search/videos/yacht/
- https://coverr.co/s/yacht
- https://pixabay.com/videos/search/yacht/

## Compression (if the file is too big)
Use https://www.freeconvert.com/video-compressor
→ set target size 4MB, resolution 1080p.
